# Projeto_Na_Unity_Escola_2JD_Turma_A
O projeto desenvolvido na Unity foi proposto pelo professor como parte das aulas práticas, com o objetivo de introduzir os alunos ao funcionamento de uma engine de desenvolvimento de jogos. Durante esse processo, foram apresentados conceitos fundamentais da criação de jogos digitais, desde a organização inicial do projeto até a implementação de interações básicas.

Ao longo das aulas, aprendemos a utilizar a interface da Unity, compreendendo elementos como a Scene, onde os objetos são posicionados; a Game, que representa a visão final do jogador; e o Inspector, responsável pela edição de propriedades dos objetos. Também foi explorado o uso de GameObjects e Components, que formam a base estrutural de qualquer projeto dentro da engine.
